# User stories
- Users first screen is the create bundle page ( frontend ) ( page )
- User can select title of bundle ( frontend )
- User can add as many links as they want ( frontend )
- User should be able to paste multiple links ( frontend ) 
- User can add a display label ( optional ) to each link ( frontend )
- User can add a note ( optional ) to each link ( frontend )
- User can create the bundle by clicking CREATE BUNDLE ( backend )
- User sees a success screen with a place to copy the link ( frontend ) ( page )
- If link isnt created a toast is shown saying error has occured ( frontend )
- If user visits that link the user can see all links of that bundle ( frontend ) ( backend ) ( page )
- Every bundle which has not been accessed in a month is deleted ( backend )
